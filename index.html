Instructions:

Using the color picker

Slide the slider and preview the value on the right hand side, Copy the value on the right hand side to one of the inputs at the 'Generator' section to apply the color

Using the 'Generator'

    Note that the input can take CSS colors such as rgb() and rgba() or just CSS colors like red, limegreen etc!

    Read more about CSS colors here
  1. Input both colors and the website will automatically apply the background for you or press the randomize button which will create a random background
  2. Press the top of the page to hide the white box (works on this page also)
  3. Screenshot and crop and make it as your new wallpaper
